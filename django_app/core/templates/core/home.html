Overview

MFA Prototype Dashboard

This Django app demonstrates one backend exposed through a simulation flow, read-only operator pages, and a small JSON API.

Use the simulation to walk through an access attempt, the backend pages to inspect configured data and session history, and the admin/API for deeper management or integration testing.
